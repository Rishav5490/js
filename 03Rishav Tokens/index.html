Tokens

Tokens are the smallest individual units in a program that are meaningful to the compiler or interpreter. They are the building blocks of a program. JavaScript has the following types of tokens:

  • Keywords
  • Identifiers
  • Operators
  • Separators
  • Literals

Example of Keywords

        
            var a = 10;
            if(a > 5){
                console.log("Hello");
            }

Example of Identifiers

        
            var a = 10;
            var b = 20;
            var sum = a + b;
            console.log(sum);

Example of Operators

        
            var a = 10;
            var b = 20;
            var sum = a + b;
            console.log(sum);

Example of Separators

        
            var a = 10;
            var b = 20;
            var sum = a + b;
            console.log(sum);

Example of Literals

        
            var a = 10;
