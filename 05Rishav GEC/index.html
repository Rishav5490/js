Global Execution Context

Global Execution Context is the default or base execution context in which JavaScript code is executed. It is created when the JavaScript engine starts to execute the code and is destroyed when the code execution is completed.

It consists of two components:

  • Global Object
  • Global Scope

Global Object: It is the top-level object in a JavaScript environment. In the browser, it is the window object. In Node.js, it is the global object.

Global Scope: It is the outermost scope in which all the variables and functions are declared. Any variable or function declared in the global scope is accessible from anywhere in the code.

Example of Global Scope

        
            var a = 10;
            function test(){
                console.log(a);
            }
            test();

Example of Local Scope

        
            function test(){
                var a = 10;
                console.log(a);
            }
            test();

Example of Local Scope

        
            function test(){
                var a = 10;
                console.log(a);
            }
            test();
            console.log(a);
