Types of Js

There are three types of Js

  • Internal Js
  • External Js
  • Inline Js

Internal Js: Js code written in the script tag in the head or body section of the HTML document is called internal Js.

External Js: Js code written in a separate file with .js extension and linked to the HTML document using the script tag is called external Js.

Inline Js: Js code written in the HTML document using the event attributes like onclick, onmouseover, onmouseout, etc is called inline Js.

Example of Internal Js

        
            <script>
                console.log("Namaste Developers..!!");
            </script>

Example of External Js

        
            <script src="script.js"></script>

Example of Inline Js

        
            <button onclick="alert('Hello Developers..!!')">Click Me</button>
