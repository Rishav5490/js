Scope is define the visibility or accessibility of variable

There are two types of scope in JS

  • Global Scope
  • Local Scope

Global Scope: Variable declared outside of function is called global scope

Local Scope: Variable declared inside of function is called local scope

Example of Global Scope

        
            var a = 10;
            function test(){
                console.log(a);
            }
            test();

Example of Local Scope

        
            function test(){
                var a = 10;
                console.log(a);
            }
            test();

Example of Local Scope

        
            function test(){
                var a = 10;
                console.log(a);
            }
            test();
            console.log(a);
